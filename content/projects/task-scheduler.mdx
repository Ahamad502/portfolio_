## Overview

Task Scheduler App is a full-featured task management solution with a beautiful user interface and powerful features to help you stay productive and organized.

## Key Features

- **Task Management**:
  - Create, edit, and delete tasks
  - Priority-based organization (High, Medium, Low)
  - Task categorization and tagging
  - Due date management
- **Smart Reminders**: 
  - Overdue detection
  - Real-time notifications
  - Smart reminder system
- **Theme Support**:
  - Dark mode for night owls
  - Light mode for day workers
  - One-click theme toggle
  - Persistent theme preference
- **Statistics Dashboard**:
  - Task completion rate
  - Overdue task count
  - Tasks by priority
  - Visual analytics
- **Data Persistence**: All tasks stored locally using Local Storage API
- **Responsive Design**: Perfect on desktop, tablet, and mobile devices

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Tailwind CSS, custom theme system
- **Icons**: Font Awesome 6.4.0
- **Storage**: Local Storage API (no backend needed)
- **Design**: Fully responsive, mobile-first approach
- **Version Control**: Git & GitHub
