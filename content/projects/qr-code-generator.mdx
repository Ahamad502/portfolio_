## Overview

QR Code Generator is a sleek, user-friendly tool for creating custom QR codes on the fly. With a beautiful gradient UI and smooth animations, generating and customizing QR codes has never been easier.

## Key Features

- **Custom QR Code Generation**: Create QR codes from any text or URL
- **Real-Time Customization**:
  - Custom foreground and background colors
  - Adjustable size (from 50 to 1000 pixels)
  - Margin control (0 to 10)
  - Error correction levels (L, M, Q, H)
- **Multiple Export Formats**: Download as PNG or SVG
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile
- **Beautiful UI**: Gradient backgrounds with smooth animations and transitions
- **No Dependencies**: Uses vanilla JavaScript with QRCode.js library

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Tailwind CSS, custom animations
- **Icons**: Font Awesome 6.4.0
- **QR Generation**: QRCode.js Library
- **Version Control**: Git & GitHub
- **Deployment**: Netlify
