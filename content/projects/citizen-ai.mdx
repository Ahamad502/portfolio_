## Overview

CitizenAi is a modern civic engagement platform designed to bridge the gap between citizens and community services. With AI-powered assistance, community news, and an easy complaint system, it helps users participate more effectively in civic matters.

## Key Features

- **AI-Powered Chatbot**: Intelligent conversational interface powered by IBM Watsonx AI for answering civic questions
- **Community News Feed**: Real-time updates on community events and local news
- **Feedback & Complaint System**: Direct channel to submit complaints and feedback via FormSpree
- **Responsive Design**: Beautiful, mobile-friendly interface built with HTML5 and CSS3
- **Flask Backend**: Robust Python backend for handling requests and AI integration

## Technologies Used

- **Backend**: Python 3, Flask
- **AI**: IBM Watsonx AI
- **Frontend**: HTML5, CSS3, Jinja2 Templates
- **Forms**: FormSpree for feedback submission
- **Deployment**: Render.com
